# Sorting Algorithms Analysis

A Python project implementing and analyzing multiple sorting algorithms including insertion sort, randomized quicksort, and merge sort.

## Overview

This project demonstrates several classic sorting algorithms while analyzing recursion behavior, comparisons, exchanges, and subproblem sizes.

The project focuses on:
- Algorithm implementation
- Recursion tracing
- Sorting efficiency
- Algorithm analysis concepts

## Algorithms Included

- Insertion Sort
- Randomized Quick Sort
- Merge Sort

## Features

- Comparison counting
- Exchange counting
- Recursive tracing
- Pivot selection visualization
- Subproblem size tracking
- Sorted output verification

## Technologies Used

- Python

## Concepts Demonstrated

- Sorting algorithms
- Divide and conquer
- Recursion
- Algorithm analysis
- Time complexity concepts
- Randomized algorithms

## How to Run

```bash
python sorting_algorithms.py
```
